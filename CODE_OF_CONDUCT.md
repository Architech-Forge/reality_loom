# Code of Conduct

Reality Loom is intended to be a serious, open, constructive project for advancing reality-oriented computing.

Participants are expected to:

- be respectful
- critique ideas, not people
- assume good faith where reasonable
- avoid harassment, abuse, or personal attacks
- keep private information private
- avoid posting secrets, user data, or sensitive materials
- respect the distinction between open foundation and private products

Maintainers may remove issues, comments, or contributions that violate these expectations.
