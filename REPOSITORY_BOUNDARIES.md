# Repository Boundaries

Reality Loom is the open foundation.

Private applications and proprietary product systems are not part of this repository.

## Public

Appropriate for this repository:

```text
Reality-Oriented Computing Codex
World Graph Engine schemas/types
World Interaction Language protocol definitions
World Definition Language examples/spec
World Intermediate Representation notes
Spatial Living Interface contracts
Reference implementation examples
Compliance fixtures
Architecture diagrams
Public essays and announcements
Deterministic test cases
```

## Private

Do not include:

```text
SoBirdi / LilBirdi application source
Sentii application source
DeckLogic application source
Autelier application source
Trader Sherpa application source
Private app UI implementation
Private cloud runtime
Private AI orchestration
Private prompts considered moat
Private admin tooling
Private avatar pipeline
Private commerce/recommendation systems
Private API keys or credentials
User data
Production logs
Production traces
Investor docs
Customer docs
Financial docs
```

## Boundary Rule

```text
Open-source the foundation.
Keep the products and moat private.
```

## Good Public Examples

```text
A tiny reference world
A deterministic WIL envelope test
A WDL-to-WGE compile example
A toy SLI projection
A sample compliance fixture
```

## Bad Public Examples

```text
A real SoBirdi styling prompt
A live product adapter
A production Supabase schema with secrets
A private app screenshot containing user data
A private admin trace
A proprietary avatar generation pipeline
```
